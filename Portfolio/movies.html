Project Purpose and Goal

A movie search app that displays movies upon typing in a specific movie name in the search box. It also displays movies randomly on page load.

Web Stack and Explanation

The webpage was built using HTML and CSS. For the movies displayed, I used an API through JavaScript that helps generate movies randomly and also by searching a particular movie. Also has a nice overview effect when you hover over a movie. Used JavaScript to also generate the color of the ratings based on the number of the rating.
