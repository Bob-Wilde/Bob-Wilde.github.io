Project Purpose and Goal

A general search app that allows users to search for anything that comes in mind.

Web Stack and Explanation

This app was built with HTML and designed with SASS/CSS. JavaScript was used to add functionality to the app by using a wikipedia API that retrieves data from thier server. The heading of each result displayed is a link that redirects to the main wikipedia page to read move about your search.
